# 💬 Zhift Feedback & Issues

This is the public repository for reporting bugs, suggesting features, and providing feedback for [Zhift](https://zhift.net).

### 🚀 How to provide feedback:
1. Click on the **[Issues](https://github.com/lokmanelouk/Zhift-Feedback/issues)** tab at the top.
2. Click the green **New Issue** button.
3. Describe your problem or suggestion clearly.

**Note:** This repository is for feedback only. The source code of Zhift is proprietary and not hosted here.
